# i-Input CSS

Simple CSS only input component 

i-input.css / v1.0.0

### Checkbox and Radio (CSS only)

![example]()

#### checkbox

````
<div class="i-check">
  <label>
    <input type="checkbox" name="name"/>
    <span>LABEL</span>
  </label>
</div>
````

#### radio
````
<div class="i-radio">
  <label>
    <input type="radio" name="name"/>
    <span>LABEL</span>
  </label>
</div>
````
