# i-Input CSS

Simple CSS only Radio and Checkbox 

i-input.css / v1.0.0

### Radio and Checkbox (CSS only)

![example](https://github.com/YSidorka/i-input/blob/master/view.jpg)

#### checkbox

````
<div class="i-check">
  <label>
    <input type="checkbox" name="name"/>
    <span>LABEL</span>
  </label>
</div>
````

#### radio
````
<div class="i-radio">
  <label>
    <input type="radio" name="name"/>
    <span>LABEL</span>
  </label>
</div>
````
